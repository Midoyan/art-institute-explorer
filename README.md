# 🎨 Art Institute Explorer

React + TypeScript app using the Art Institute of Chicago API. Search artworks, save favourites to a gallery, and add notes — with Zod validation and localStorage persistence.

## Features
- Search artworks (AIC API)
- Validate API data with Zod
- Save/remove favourites (localStorage)
- Add/edit notes per artwork

## Tech
React • TypeScript • Vite • Zod

## Run locally
```bash
npm install
npm run dev
